# Changelog

All notable changes to GPA Suite will be documented here.

## [1.0.0] - 2026-05-28

### 🎉 Initial Release

### Added

- Single Semester GPA Calculator with subject-level input
- Semesters by SGPA tab for quick CGPA calculation
- Multi-Semester tab with full subject detail across semesters
- Target CGPA Planner to find required future GPA
- GPA Scale Editor with custom ranges and presets
- Light / Dark / Blue theme support
- CSV Import and Export functionality
- Save/Load project feature in Multi-Semester tab
- Fill Demo button for quick testing
- Responsive status bar with live SGPA/CGPA display
